### Features

- A multi-input field for Laravel Nova.
- Also supports dark mode.

### Installation

Install this package in a Laravel Nova project via Composer:

```bash
composer require jshxl/multi-input
```

### Requirements

- `php: ^8.1`
- `laravel/framework: ^10.10`
- `laravel/nova: ~4.0`
- `jshxl/primevue: ^1.3.3`
